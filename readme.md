1. 1-4 SEAT SYSTEM
  학생들의 자리를 무작위로 배치할 수 있는 C 프로그램이다. 학생 등록, 랜덤 배치, 자리 고정 및 교환, 학생 추가 및 제거가 가능하다.

2. 프로그램 사용 방법
  메인 메뉴에서 메뉴를 선택하여 번호만 입력한다.
  0) 학생 등록: 등록할 전체 학생 수를 입력받은 후(최대 40명), 학생의 이름을 엔터 간격으로 입력받는다. 학생 수는 6의 배수가 됨을 권장한다. 모든 학생의 이름이 한글 3글자여야 하기 때문에 이름이 2글자인 학생의 이름은 가운데를 한 칸 띄우고, 이름이 4글자 이상인 학생의 이름은 3글자까지만 입력한다. 동일한 이름은 중복 등록할 수 없다.
  1) 1-4학생 자동 등록: 미리 저장된 1-4반 학생 명단을 별도의 입력 없이 자동으로 등록한다.
  2) 자리 랜덤 배치: 고정된 학생을 제외하고 등록된 학생들을 랜덤으로 배열한 후 출력한다.
  3) 현재 자리 출력: 저장된 현재 자리 배치표를 불러와 출력한다.
  4) 자리 상호 교환: 자리를 서로 교환할 학생의 자리 번호 2개를 입력받은 후, 두 사람의 자리가 교환된 자리 배치표를 출력한다. 고정된 자리는 교환할 수 없다.
  5) 자리 고정: 고정할 학생 수를 입력받은 후, 이어서 고정할 자리와 학생 수를 입력받는다. 이때 자리를 고정할 때마다 이전에 했던 고정 기록은 초기화된다. 학생 자리를 고정하면 자리를 랜덤 배치할 때 반영되어 지정한 자리를 유지한다.
  6) 학생 추가: 학생 1명을 추가로 입력받는다. 총 인원 수는 최대 40명으로, 그 이상으로는 등록할 수 없다. 이미 등록된 학생은 추가할 수 없다.
  7) 학생 제거: 명단에 있던 학생 1명을 제거한다. 빈 자리는 제거된 학생의 뒤에 있는 학생들이 한 칸씩 왼쪽 또는 앞으로 이동한다.
  8) 자리 배치도 내보내기: 현재 자리 배치도를 txt 파일로 내보낸다.
  9) 프로그램 종료: 실행되고 있던 프로그램을 종료한다. 모든 기록은 사라진다. 

3. 버그 및 디버그
  현재까지의 제한 사항
  - 최대 학생 수는 40명이다.
  - 학생 이름은 최대 19글자까지 입력 가능하다.
  - 새로운 자리 고정을 설정하면 기존의 고정 정보는 초기화된다.
  - skyman.txt 파일이 없으면 이스터에그를 실행할 수 없다.
